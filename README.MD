# Описание  проекта 
Получает данные о компаниях и вакансиях с сайта hh.ru, спроектирует таблицу в БД PostgreSQL и загрузит полученные данные в созданную таблицу.

## Исползуемые технологии 
  * python #(3.11) 
  * requests #(2.31.0)
  * psycopg2==2.9.9
